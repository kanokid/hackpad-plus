# Hackpad+
This is a hackpad that I made for Blueprint.
* It uses the Seeed Studio XIAO RP2040
* It has per-key shine through RGB
* It has 9 MX style switches and a rotary encoder for media controls/volume.
* The keys are through a 
* It uses KMK configured through pog for the firmware
